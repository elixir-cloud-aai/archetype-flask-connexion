Opinionated set of tools/utils for quickly developing OpenAPI-based
microservices with Flask and Connexion.

## Usage

Install with `pip`:

```bash
pip install foca
```

Import in your code! For example:

```bash
from foca.config.config_parser import YAMLConfigParser
```
